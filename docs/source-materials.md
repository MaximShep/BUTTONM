# Source Materials

В проекте есть два исходных PDF-файла:

- пример рекламного брифа
- эталонные сценарии, написанные по этому брифу

Они остаются файлами в корне репозитория. Код обращается к ним через `lib/files/source-materials.ts`, чтобы позже их можно было использовать для извлечения текста, обучения правил стиля или сравнения качества генерации.

Содержимое PDF не копируется в код и не хранится в seed-данных.
